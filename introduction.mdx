# Introduction to Chatwoot

Chatwoot is the modern, open-source, and self-hosted customer support platform designed to help businesses deliver exceptional customer support experiences. Built for scale and flexibility, Chatwoot gives you full control over your customer data while providing powerful tools to manage conversations across channels.

<Note>
Chatwoot is an open-source alternative to Intercom, Zendesk, Salesforce Service Cloud, and other commercial customer support platforms.
</Note>

## What is Chatwoot?

Chatwoot centralizes all your customer conversations into one powerful platform, enabling support teams to:

- Manage customer conversations across multiple channels from a unified inbox
- Collaborate with team members using internal notes and mentions
- Automate repetitive tasks with rules and AI-powered assistance
- Track performance with detailed reports and analytics
- Build self-service help centers to reduce support load

## Key Capabilities

### Omnichannel Support Desk

Chatwoot centralizes all customer conversations into one powerful inbox, no matter where your customers reach out from. It supports:

- **Website Live Chat** - Embed a customizable chat widget on your website
- **Email** - Connect your support email and handle messages seamlessly
- **Social Media** - Facebook, Instagram, Twitter integration
- **Messaging Apps** - WhatsApp, Telegram, Line support
- **SMS** - Twilio SMS integration for text message support
- **API Channel** - Build custom integrations with the API channel

### Collaboration & Productivity

Empower your team with tools designed for efficient collaboration:

- **Private Notes** - Add internal notes with @mentions for team discussions
- **Labels** - Organize and categorize conversations for better tracking
- **Canned Responses** - Save time with pre-written responses for common questions
- **Auto-Assignment** - Automatically route conversations based on agent availability
- **Teams** - Organize agents into teams for better workflow management
- **Keyboard Shortcuts** - Navigate faster with command bar and shortcuts
- **Custom Views & Filters** - Create personalized views for inbox organization
- **Business Hours** - Set working hours and auto-responders

### Customer Data Management

Build comprehensive customer profiles:

- **Contact Management** - Store customer profiles with interaction history
- **Custom Attributes** - Add custom fields to store additional data
- **Contact Segments** - Create targeted segments for personalized communication
- **Pre-Chat Forms** - Collect information before conversations start
- **Campaigns** - Proactively engage customers with targeted messages

### Help Center Portal

Enable customer self-service:

- Publish help articles, FAQs, and guides
- Organize content with categories and sections
- Multi-language support for global audiences
- SEO-friendly article URLs
- Reduce repetitive queries by empowering customers to find answers

### Reports & Analytics

Measure performance and gain insights:

- **Live View** - Monitor ongoing conversations in real-time
- **Conversation Reports** - Track conversation volume and trends
- **Agent Reports** - Measure individual agent performance
- **CSAT Reports** - Monitor customer satisfaction scores
- **Inbox, Label, and Team Reports** - Get operational visibility
- **Downloadable Reports** - Export data for offline analysis

### Integrations

Extend Chatwoot with powerful integrations:

- **Slack** - Manage conversations directly from Slack
- **Dialogflow** - Add chatbot automation capabilities
- **Shopify** - View and manage customer orders within Chatwoot
- **Google Translate** - Translate customer messages in real-time
- **Linear** - Create and manage tickets seamlessly
- **Dashboard Apps** - Embed internal tools within Chatwoot
- **Webhooks** - Build custom integrations with webhook events

## Architecture Overview

Chatwoot is built with modern technologies:

- **Backend**: Ruby on Rails with PostgreSQL database
- **Frontend**: Vue.js 3 with Composition API and Tailwind CSS
- **Real-time**: WebSocket support via Action Cable
- **Cache**: Redis for caching and background jobs
- **Storage**: Active Storage for file attachments

<Note>
Chatwoot follows a modular architecture that separates channels (integrations), inboxes (conversation containers), and conversations (customer interactions).
</Note>
